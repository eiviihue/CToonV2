# CToon - Comic Reading Website

This project is a full-featured, responsive Comic Reading Website built with JSP (Jakarta Server Pages) for deployment on Railway via GitHub.

## Features
- CRUD operations for comics, chapters, comments, ratings, bookmarks, and user profiles
- Authentication (login/signup with email/username, guest login)
- Comic-chapter viewing with navigation and chapter-specific comments
- Comic information page with cover, description, comments, average rating (5-star), category, bookmark and rating buttons
- User restrictions for commenting/rating/bookmarking
- Comic search/filter by category/views/updates
- Bookmark page and profile page (edit profile/photo/password)
- Navbar with search and navigation
- Home page with recent updates and ratings
- Dark/Light mode
- Asset folders for comic covers and chapter images

## Structure
- `src/` - Java source files (Servlets, Models, Controllers)
- `web/` - JSP files, static assets (CSS, JS, images)
- `assets/cover/` - Comic covers
- `assets/comics/` - Comic chapter images
- `.github/` - Project instructions

## Deployment
- Designed for Railway deployment via GitHub
